## Content production guardrails

Every piece of content passes through:

1. **Anti-slop kill list** (`copywriting/anti-slop-kill-list.md`) — banned words, banned phrase patterns, sentence-structure tells
2. **Three-line sanity check** — would the business owner say this? Could this be about any other painter? Does it match what the reader was thinking?
3. **Kitchen-table test** — read aloud, no jargon, 5th–6th grade reading level (Hemingway 5–6)
4. **`claude-seo:seo-content` E-E-A-T scoring** — must pass before publish
5. **Fact verification** — no fabricated specifics; the "why that number" test from the anti-slop list

For suburb variants specifically: do not write 48 versions of the same paragraph with the suburb name swapped. If a paragraph could apply to any suburb, cut it.
