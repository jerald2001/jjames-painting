# Website Build 
I want a very dummy-proof way of building new websites for Tetelestai - so that newly trained devs/workers can easily build websites that look good and functional.

## Needed Plugin/MCP/Skill
- superpowers
- playwright
- agent-browser
- seo

## Default files
- **CLAUDE.md** very generic so it'll cite most of the details from the brainstorming and client-brief.md
- **PLAN.md** having the build in phases and 

## Process
1. dev to create a client-brief.md with all the info of the client - can be automated in Claude as well.
2. upload assets like brand guide or even just the logo so it'll get their client's colors
3. begin session and start with brainstorming so claude can ask the dev clarifying questions
4. QA visually
5. Code Review
5. Deploy 



# Landing Page Build Plugin
A plugin that will automatedly create the landing pages for Smart Local

## Feature
I can install to an existing project so that devs can easily build the landing pages for our websites.

## Process
1. Run a command in the terminal
2. Upload the copy/content in an .md file
3. Identify the Pillar page or main city
4. List all the suburbs
5. Claude will do the rest
6. QA
