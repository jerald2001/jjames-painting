# Approved Image Patterns

Used in Phase 3 (Build) and Phase 4 (Content). Referenced from [`CLAUDE.md`](../CLAUDE.md) > Image & Visual Showcase Strategy. See [`PLAN.md`](../PLAN.md) for checklist items.

Approved image layout patterns for inserting images into website pages.

| Pattern | Description | Best For |
|---|---|---|
| **Bento grid** | Asymmetric grid with varied cell sizes (1x1, 1x2, 2x1, 2x2). No uniform squares. | Service overviews, portfolio showcases, feature highlights |
| **Staggered/offset** | Image bleeds past the content column or overlaps the adjacent section. Text and image are not neatly aligned. | Alternating text+image sections, service pages |
| **Full-bleed break** | Full-width image spanning edge to edge with text overlay or caption bar. Acts as a visual "chapter break" between content sections. | Between major sections, page rhythm |
| **Floating accent** | Small image or detail shot positioned asymmetrically alongside text. Uses negative margin or absolute positioning to break the grid. | Supporting details, texture shots |
| **Parallax/scroll reveal** | Image stays fixed or scales subtly on scroll. CSS-only (`background-attachment: fixed` or `transform` on scroll). No JS animation libraries. | Hero sections, mid-page impact moments |
| **Collage/overlap** | 2-3 images overlapping at slight angles or offsets with shadow separation between layers. | Testimonials, about pages, team sections |
| **Before/after split** | Side-by-side or slider comparison showing transformation. | Service results, project showcases |
| **Inline editorial** | Image sits within the text column with text wrapping around it, magazine-style. Uses `float` or CSS shapes. | Blog posts, long-form content pages |
| **Diagonal/clip-path reveal** | Image revealed through an angled clip-path or diagonal cut, creating visual tension with the surrounding content. | Hero sections, feature highlights |
| **Card with bleed** | Image extends beyond its card container, breaking the expected boundary. | Testimonials, service cards, pricing sections |
